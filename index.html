Tech Stack — Why These Libraries?

Flask

Lightweight micro-framework. No Django bloat — exactly what's needed: fast, reliable REST API routing.

subprocess

Most critical library. Securely spawns OS processes, executes Nmap binary, captures stdout/stderr without crashing the server.

os

Dynamically fetches env variables like $PORT. Best practice for cloud deployments with dynamic port assignment.

Gunicorn (WSGI)

Flask's dev server isn't production-ready. Gunicorn handles concurrency + --timeout 100 prevents hangups during deep scans.

Cloud Deployment
Challenges Overcome

Challenge 1

The Raw Socket Restriction

Cloud providers run unprivileged Docker containers, blocking standard SYN stealth scans (-sS).

Solution

Engineered TCP Connect scans (-sT) — fully compliant with unprivileged containers while maintaining deep reconnaissance capability.

Challenge 2

The Timeout Battle

Deep intelligence takes time, but cloud servers kill long-running processes. Balancing depth vs. stability was critical.

Solution

Combined fast flags -F -T4 --max-retries 1 with Gunicorn --timeout 100 override for stable deep scans.
